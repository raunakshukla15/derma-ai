Problem Statement : Identifying skin diseases early is difficult without access to dermatologists. Many people lack quick, reliable tools to get an initial understanding of skin conditions.

Solution Overview : Derma AI is an AI-powered web application that analyzes images of skin conditions to predict the most likely disease. Users can upload an image and receive the disease name, symptoms, description, treatment guidance, severity level, and an AI-assisted recommendation. This tool is intended for educational and awareness purposes only.

Tech Stack : 
Frontend: HTML, CSS, JavaScript
Backend: Python, Flask
AI/ML: PyTorch (CNN-based image classification) & Agentic Doctor
Data Processing: NumPy, Pandas
Image Processing: Pillow, OpenCV
Deployment: Docker, Hugging Face Spaces

Setup Instructions :

for : Local

Clone or download the repository
Create and activate a virtual environment
Install dependencies: pip install -r requirements.txt
Run the application in Terminal : python app.py
Open http://127.0.0.1:7860 in your browser

Disclaimer : This application is not a medical diagnosis tool. Always consult a qualified healthcare professional for medical advice.
